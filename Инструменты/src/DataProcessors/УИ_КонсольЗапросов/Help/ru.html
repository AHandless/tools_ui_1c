Обработка "Униформальная консоль запросов" предназначена для составления и исполнения запросов 1С:Предприятия в обычных и управляемых формах максимально единообразным способом. Предназначена для всех, но существует некоторая корреляция полезности обработки с интеллектом и знанием языка SQL. Диалог обработки разбит на три основных блока.

Дерево запросов. В левой верхней части обработки располагается дерево запросов, составленных пользователем. Возможно добавление, удаление, перемещение элементов этого списка. По двойному щелчку мыши по строке дерева происходит выполнение выбранного запроса. Управление деревом осуществляется с помощью блока кнопок командной панели обработки.
Созданное дерево запросов можно сохранить в файле с расширением sel. В обработке предусмотрен режим автоматического сохранения запросов. Для этого необходимо по кнопке командной панели вызвать диалог настройки автосохранения, включить режим автосохранения и установить желаемый интервал автоматического сохранения запросов в файл.

Текст запроса. В правой верхней части обработки располагается непосредственно текст запроса. Отображаемый запрос зависит от текущей строки дерева запросов. Если в тексте запросов выделить часть текста, то выполняться будет только выделенная часть, а не весь написанный текст запроса (если данная опция указана в настройках обработки). Над текстом запроса располагаются кнопки "Выполнить" и "Параметры".
По кнопке "Параметры" вызывается отдельное окно, в котором можно задать список параметров запроса или заполнить используемые параметры из текста запроса по кнопке "Получить из запроса.
В первой колонке указывается имя параметра. Во второй колонке указывается тип параметра (значение, список значений или таблица значений). В третьей колонке указывается собственно параметр запроса.

Результат запроса. В нижней части диалога располагается результат выполнения запроса в виде таблицы. Ячейки данной таблицы кликабельны, по клику открываются соотвествующая типу значений форма просмотра. Над содержимым результата выполнения запроса можно выполнить простые действия с помощью пункта контекстного меню "Произвольный код". Над результатом запроса располагается информационная строка, которая показывает время выполнения запроса в секундах и количество строк в результате запроса.
